# Nursly — The Quiet Blog

A calm, distraction-free social blogging platform.

## Overview

The Quiet Blog is a clean, modern social blogging platform where people can share thoughts, stories, opinions, photos, and daily experiences in a peaceful, distraction-free environment.

**Tagline:** Share Thoughts. Keep It Calm.

This repository hosts the source for the platform. See the pull requests for the full application.
